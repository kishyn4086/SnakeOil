Snake Oil is a computer g­ame inspired by a mobile Snake game that originally appeared on a Nokia device. This game is a project that addresses the issue of digital consumption and its environmental impact from the perspective of designers or developers. To enhance the visual experience, the size of game files tends to increase as more resources are utilized for graphics, sound, video, textures, scripts, and other elements. However, at some point, the fascination with simplicity has faded. This project is designed to demonstrate how a lightweight viewing experience can still provide a better user experience by removing unnecessary features and optimizing the site's performance in simplified conditions.
